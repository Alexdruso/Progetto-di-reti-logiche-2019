# Test Bench generator

Generate VHDL test benches for the project.

## Usage

```
python tb_generator.py
```

The generator outputs the `.vhd` test benches in the `out/` folder.

You can customize the test benches parameters by editing the `test_benches` list
inside `tb_generator.py`

## Compatibility

The script was tested with Python 3.8.1 on Windows 10.
